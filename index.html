Functions

List Functions

  • list.append(x): Adds an item x to the end of the list.
  • list.extend(iterable): Extends the list by appending elements from an iterable.
  • list.insert(index, x): Inserts an item x at a specified position index.
  • list.remove(x): Removes the first occurrence of an item x from the list. Raises a ValueError if the item is not found.
  • list.pop([index]): Removes and returns the item at the specified position index in the list. If no index is specified, it removes and returns the last item.
  • list.clear(): Removes all items from the list.
  • list.index(x[, start[, end]]): Returns the index of the first occurrence of an item x. Raises a ValueError if the item is not found.
  • list.count(x): Returns the number of occurrences of an item x in the list.
  • list.sort(key=None, reverse=False): Sorts the items of the list in place.
  • list.reverse(): Reverses the elements of the list in place.
  • list.copy(): Returns a shallow copy of the list.

Tuple Functions

  • tuple.count(value): Returns the number of times value appears in the tuple.
  • tuple.index(value[, start[, end]]): Returns the index of the first occurrence of value. Raises a ValueError if the value is not found.

Set Functions

  • set.add(elem): Adds an element elem to the set. If the element is already present, it does nothing.
  • set.clear(): Removes all elements from the set.
  • set.copy(): Returns a shallow copy of the set.
  • set.discard(elem): Removes an element elem from the set if it is present. Does nothing if the element is not found.
  • set.remove(elem): Removes an element elem from the set. Raises a KeyError if the element is not found.
  • set.pop(): Removes and returns an arbitrary element from the set. Raises a KeyError if the set is empty.
  • set.union(*others): Returns a new set with elements from the set and all others.
  • set.intersection(*others): Returns a new set with elements common to the set and all others.
  • set.difference(*others): Returns a new set with elements in the set that are not in the others.
  • set.symmetric_difference(other): Returns a new set with elements in either the set or other but not in both.
  • set.issubset(other): Returns True if all elements of the set are in other. Otherwise, returns False.
  • set.issuperset(other): Returns True if all elements of other are in the set. Otherwise, returns False.
  • set.isdisjoint(other): Returns True if the set has no elements in common with other. Otherwise, returns False.
  • set.update(*others): Updates the set with elements from others. This is equivalent to set = set.union(*others).
  • set.intersection_update(*others): Updates the set with the intersection of itself and others. This is equivalent to set = set.intersection(*others).
  • set.difference_update(*others): Updates the set by removing elements found in others. This is equivalent to set = set.difference(*others).
  • set.symmetric_difference_update(other): Updates the set with elements in either the set or other but not in both. This is equivalent to set = set.symmetric_difference(other).

Dictionary Functions

  • dict.clear(): Removes all items from the dictionary.
  • dict.copy(): Returns a shallow copy of the dictionary.
  • dict.fromkeys(seq[, value]): Creates a new dictionary with keys from seq and values set to value (or None if not provided).
  • dict.get(key[, default]): Returns the value for the specified key if the key is in the dictionary; otherwise, returns default (or None if not provided).
  • dict.items(): Returns a view object that displays a list of a dictionary's key-value tuple pairs.
  • dict.keys(): Returns a view object that displays a list of all the keys in the dictionary.
  • dict.pop(key[, default]): Removes the specified key and returns the corresponding value. If the key is not found and default is not provided, raises a KeyError.
  • dict.popitem(): Removes and returns the last key-value pair as a tuple. Raises a KeyError if the dictionary is empty.
  • dict.setdefault(key[, default]): Returns the value for the specified key. If the key is not present, inserts the key with a value of default and returns default.
  • dict.update([other]): Updates the dictionary with the key-value pairs from other, overwriting existing keys.

String Functions

  • str.upper(): Converts all characters in the string to uppercase.
  • str.lower(): Converts all characters in the string to lowercase.
  • str.capitalize(): Capitalizes the first character of the string and lowercases all other characters.
  • str.title(): Capitalizes the first character of each word in the string.
  • str.strip([chars]): Removes leading and trailing whitespace or specified characters.
  • str.lstrip([chars]): Removes leading whitespace or specified characters.
  • str.rstrip([chars]): Removes trailing whitespace or specified characters.
  • str.find(substring[, start[, end]]): Returns the lowest index of the substring if it is found.
